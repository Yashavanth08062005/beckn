                          TROUBLESHOOTING
================================================================================

If services stop running:

  Kill all services:
  $ pkill -f "mock-onix-adapter"
  $ pkill -f "npm start"
  $ pkill -f "npm run dev"

  Start Mock ONIX:
  $ cd bap-travel-discovery
  $ nohup node mock-onix-adapter.js > mock-onix.log 2>&1 &

  Start BAP:
  $ npm start 2>&1 &

  Start Frontend (new terminal):
  $ cd frontend-travel-discovery
  $ npm run dev

  Check logs:
  $ tail -f mock-onix.log
